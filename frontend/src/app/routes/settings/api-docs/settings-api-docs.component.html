API 使用文档

link API 端点
POST {{ apiBaseUrl }}/api/image/upload
vpn_key 认证方式

在请求头中添加 Authorization 字段：

Authorization: Api-Key YOUR_API_KEY

warning 注意：前缀是 Api-Key（带空格），不是 Bearer

info 请在 API密钥 页面创建和管理您的 API 密钥

terminal cURL 示例 
curl -X POST "{{ apiBaseUrl }}/api/image/upload" \
  -H "Authorization: Api-Key YOUR_API_KEY" \
  -F "image=@/path/to/image.png"
 code Python 示例 
import requests

url = "{{ apiBaseUrl }}/api/image/upload"
headers = {{ '{' }}"Authorization": "Api-Key YOUR_API_KEY"{{ '}' }}
files = {{ '{' }}"image": open("image.png", "rb"){{ '}' }}

response = requests.post(url, headers=headers, files=files)
print(response.json())

 help 常见问题 支持哪些图片格式？

支持 JPEG、PNG、GIF、WebP、BMP 等常见图片格式。

 图片大小限制是多少？

默认最大上传大小为 128MB，具体限制取决于服务器配置。

 如何获取上传后的图片链接？

上传成功后，使用返回的 date_path 和 id 拼接链接：{{ apiBaseUrl }}/i/[date_path]/[id]

例如：{{ apiBaseUrl }}/i/20251130/abc123
